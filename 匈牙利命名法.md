# 匈牙命名法

| 型別 | 名稱 | 字首 | 用法 |
|---|---|---|---|
| Short | 短整數 | sh | shVarname |
| Int | 整數 | n | nVarname |
| Long | 長整數 | l | lVarname |
| Float | 浮點數 | f |  fVarname |
| Double | 倍精度浮點數 | d | dVarname |
| Char | 字元 | c | cVarname |
| String | 字串 | s | sVarname |
| Boolean | 布林 | b | bVarname |
